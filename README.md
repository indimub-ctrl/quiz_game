# Python Quiz Game

A simple interactive quiz game built using Python.

## Features
- Easy and Hard difficulty levels
- Randomized (shuffled) questions
- Score tracking
- Case-insensitive answers
- Instant feedback after each question

## Technologies Used
- Python
- Built-in modules (`random`)

## How to Run
1. Make sure Python is installed
2. Download or clone the repository
3. Run the program:

   python quiz_game.py

## Example
Choose difficulty (easy/hard): easy  
2 + 2? 4  
Correct! 🎉
